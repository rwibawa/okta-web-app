## Add git submodules
```bash
$ mkdir okta-web-app
$ cd okta-web-app

$ git init
$ vi README.md

$ git submodule add git@github.com:rwibawa/okta-web-app-ui.git okta-web-app-ui
$ git submodule add git@github.com:rwibawa/okta-web-app-backend.git okta-web-app-backend
```
